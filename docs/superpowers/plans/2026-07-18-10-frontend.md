# Frontend Implementation Plan (Plan 10 of 11)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** React + Vite SPA with app-shell, admin-portal, client-portal, and shared ui-kit. All UI work must invoke the `ui-ux-pro-max` skill before component implementation.

**Architecture:** One Vite React app with role-gated routes. JWT stored in memory + httpOnly refresh cookie (or localStorage for v1 simplicity with XSS caveats documented). Tenant slug sent as `X-Tenant-Slug`. Free/OSS stack only.

**Tech Stack (free/OSS):** React 18, Vite, React Router, TanStack Query, Vitest, React Testing Library, Playwright, free icon set (lucide-react), free CSS (plain CSS modules or free Tailwind).

**Depends on:** Plans 1вЂ“9 (API surfaces).
