# AI Service Implementation Plan (Plan 8 of 11)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up the FastAPI AI microservice with LangChain provider factory (Ollama free primary for local; OpenAI optional), pgvector embeddings, health endpoints, JWT shared with Django, and optional free-tier LangSmith tracing.

**Architecture:** Separate `ai_service/` package (FastAPI) containerized alongside Django. Shared Postgres with `pgvector` extension. Provider cascade: **Ollama (free, self-hosted) → HuggingFace local embeddings (free) → OpenAI (optional paid, disabled by default)**. LangSmith Developer free tier is optional via env flag.

**Tech Stack (free/OSS by default):**
- FastAPI, uvicorn
- LangChain, langchain-core, langchain-community, langchain-experimental, langgraph, langchain-postgres, langchain-ollama
- pgvector (Postgres extension, free)
- Ollama (self-hosted, free)
- sentence-transformers / HuggingFace embeddings (free, local)
- OpenAI package present but **disabled unless `AI_ENABLE_OPENAI=1`**
- LangSmith optional (`LANGCHAIN_TRACING_V2=true` only if user opts in)

**Depends on:** Plan 1 (shared JWT secret + Postgres).

---

## Free/OSS hard rules for this plan

1. Default LLM = Ollama (`llama3.1` or `qwen2.5`).
2. Default embeddings = `BAAI/bge-small-en` via HuggingFace (local) OR Ollama embeddings.
3. OpenAI is opt-in only (`AI_ENABLE_OPENAI=1` + `OPENAI_API_KEY`).
4. LangSmith is opt-in only (free Developer tier, SaaS exception from design).
5. No paid vector DB (pgvector only).

```python
# ai_service/auth.py
"""JWT verification shared with Django."""
from fastapi import Depends, HTTPException
from fastapi.security import HTTPAuthorizationCredentials, HTTPBearer
from jose import JWTError, jwt

from ai_service.config import settings

_bearer = HTTPBearer()


def current_user(
    creds: HTTPAuthorizationCredentials = Depends(_bearer),
) -> dict:
    """Decode and return JWT claims.

    Args:
        creds: Bearer token.

    Returns:
        Claims dict including user_id and tenant_id if present.

    Raises:
        HTTPException: 401 on invalid token.
    """
    try:
        return jwt.decode(
            creds.credentials,
            settings.jwt_secret,
            algorithms=["HS256"],
        )
    except JWTError as exc:
        raise HTTPException(status_code=401, detail="invalid token") from exc
```

### Task 3: LLM factory (Ollama primary, OpenAI opt-in)

```python
# ai_service/llm/factory.py
"""Chat model factory with free-first cascade."""
import httpx
from langchain_core.language_models.chat_models import BaseChatModel
from langchain_ollama import ChatOllama

from ai_service.config import settings


def ollama_reachable() -> bool:
    """Return True if Ollama responds to /api/tags.

    Returns:
        Reachability flag.
    """
    try:
        r = httpx.get(f"{settings.ollama_base_url}/api/tags", timeout=2.0)
        return r.status_code == 200
    except httpx.HTTPError:
        return False


def get_chat_model() -> BaseChatModel:
    """Return the configured chat model.

    Preference order:
    1. Ollama (free, self-hosted) if reachable.
    2. OpenAI only if ``AI_ENABLE_OPENAI=1`` and key set.

    Returns:
        A LangChain chat model.

    Raises:
        RuntimeError: if no provider is available.
    """
    if ollama_reachable():
        return ChatOllama(
            base_url=settings.ollama_base_url,
            model=settings.ollama_model,
        )
    if settings.enable_openai and settings.openai_api_key:
        from langchain_openai import ChatOpenAI
        return ChatOpenAI(
            api_key=settings.openai_api_key,
            model="gpt-4o-mini",
        )
    raise RuntimeError("No LLM provider available (start Ollama or enable OpenAI)")
```

### Task 4: Embeddings cascade (HF free → Ollama → optional OpenAI)

```python
# ai_service/llm/embeddings.py
"""Embedding model factory, free-first."""
from langchain_core.embeddings import Embeddings
from langchain_community.embeddings import HuggingFaceEmbeddings
from langchain_ollama import OllamaEmbeddings

from ai_service.config import settings
from ai_service.llm.factory import ollama_reachable


def get_embeddings() -> Embeddings:
    """Return embeddings model.

    Preference:
    1. HuggingFace local ``BAAI/bge-small-en`` (free).
    2. Ollama embeddings if HF unavailable and Ollama up.
    3. OpenAI embeddings only if ``AI_ENABLE_OPENAI=1``.

    Returns:
        Embeddings instance.
    """
    try:
        return HuggingFaceEmbeddings(model_name=settings.embedding_model)
    except Exception:  # noqa: BLE001
        pass
    if ollama_reachable():
        return OllamaEmbeddings(
            base_url=settings.ollama_base_url, model=settings.ollama_model
        )
    if settings.enable_openai and settings.openai_api_key:
        from langchain_openai import OpenAIEmbeddings
        return OpenAIEmbeddings(api_key=settings.openai_api_key)
    raise RuntimeError("No embeddings provider available")
```
